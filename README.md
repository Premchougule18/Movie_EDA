# Movie EDA

This project contains the exploring a movie dataset and visualizing key trends in the data.

## Files
- `eda_movies.ipynb` - Main exploratory data analysis notebook
- `movies_dataset.csv` - Movie dataset used in the notebook

## What this notebook covers
- Loading and inspecting the dataset
- Checking data types, shape, and missing values
- Exploring distributions of numeric and categorical features
- Visualizing relationships such as budget vs. box office performance
- Comparing genre trends and actor ratings

## How to run
1. Open `eda_movies.ipynb` in Jupyter Notebook or JupyterLab.
2. Run the cells in order to reproduce the analysis.
3. Make sure `movies_dataset.csv` is in the same folder as the notebook.
